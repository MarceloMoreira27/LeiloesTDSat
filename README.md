Nome do projeto: LeiloesTDSat

Explicação sobre o que se trata o projeto:
  Este projeto consiste num sistema informatizado para a gestão de rotinas de uma casa de leilões. 
O sistema permite o registo de produtos, controlo de lances e monitorização de vendas, garantindo a rastreabilidade e segurança dos dados.

Tecnologias utilizadas: Java e MySQL
