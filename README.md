# LeiloesTDSat 🔨

Este é um sistema simples para gerenciar uma **Casa de Leilões**. Com ele, é possível cadastrar produtos, registrar lances e ver o que já foi vendido.

## 📝 O que o projeto faz?
O projeto ajuda a organizar o trabalho de um leiloeiro, trocando o papel pelo computador:
* **Cadastra produtos:** Salva o nome e o valor inicial do item.
* **Controla lances:** Registra quem deu o maior valor.
* **Acompanha vendas:** Mostra o que está disponível ou vendido.
* **Segurança:** Garante que os dados não sejam perdidos.

## 🛠️ Tecnologias usadas
* **Java:** A linguagem que faz o programa funcionar.
* **MySQL:** O banco de dados onde as informações ficam guardadas.
